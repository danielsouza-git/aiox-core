# Brand Health Checklist

Quality checklist for brand perception audits, touchpoint audits, and consistency scoring.

## Perception Assessment

- [ ] **Intended perception documented** - Target brand attributes with ratings defined
- [ ] **All channels audited** - Every specified channel assessed
- [ ] **Consistent methodology** - Same rating scale across all channels
- [ ] **Evidence captured** - Specific examples support each rating
- [ ] **Gap calculation correct** - Intended minus actual calculated accurately

## Touchpoint Coverage

- [ ] **All touchpoints discovered** - Comprehensive discovery completed
- [ ] **Journey stages mapped** - Touchpoints linked to customer journey
- [ ] **Ownership assigned** - Each touchpoint has a responsible team/person
- [ ] **Digital touchpoints complete** - Website, social, email, ads covered
- [ ] **Physical touchpoints covered** - If applicable, packaging, signage, materials audited

## Scoring Objectivity

- [ ] **Scoring criteria defined** - Clear definitions for each score level
- [ ] **Consistent application** - Same criteria applied across all touchpoints
- [ ] **Multiple assessors** - Where possible, multiple perspectives gathered
- [ ] **Bias mitigation** - Assessor acknowledged potential biases
- [ ] **Evidence-backed** - Each score tied to specific observations

## Recommendations Quality

- [ ] **Prioritized by impact** - Recommendations ranked by customer impact
- [ ] **Effort estimated** - Quick/Moderate/Major effort labeled
- [ ] **Actionable** - Each recommendation has a clear owner and next step
- [ ] **Measurable outcomes** - Success metrics defined for improvements
- [ ] **Timeline provided** - Implementation phases with dates

## Completeness

- [ ] **Executive summary** - Key findings summarized concisely
- [ ] **Quantified scores** - Overall score (0-100) calculated
- [ ] **Dimension breakdown** - Per-dimension scores provided
- [ ] **Trend data** - Compared to previous audit if available
- [ ] **Methodology documented** - How the audit was conducted

## Delivery

- [ ] **Report structured** - Follows standard template
- [ ] **Visualizations included** - Score breakdowns shown graphically
- [ ] **No placeholder content** - All sections completed
- [ ] **Proofread** - No spelling or formatting errors
- [ ] **Standalone document** - Reader needs no additional context
