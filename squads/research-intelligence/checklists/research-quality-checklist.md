# Research Quality Checklist

Universal quality checklist for all research deliverables.

## Data Quality

- [ ] **Sources cited** - Every data point has a named source with date
- [ ] **Data recency** - All data less than 12 months old (or explicitly flagged as older)
- [ ] **Source diversity** - Minimum 3 independent sources for key metrics
- [ ] **Confidence labeling** - Each estimate labeled High/Medium/Low confidence
- [ ] **Methodology documented** - Research approach clearly described
- [ ] **No single-source claims** - Critical findings backed by multiple sources

## Research Rigor

- [ ] **Scope defined** - Research boundaries clearly stated
- [ ] **Assumptions documented** - All assumptions listed explicitly
- [ ] **Fact vs. inference** - Clear distinction between confirmed data and analyst interpretation
- [ ] **Limitations acknowledged** - Data gaps and limitations noted
- [ ] **Bias check** - Sources checked for potential bias
- [ ] **Cross-validation** - Key findings validated across sources

## Structure & Clarity

- [ ] **Executive summary** - 2-3 paragraph overview of key findings
- [ ] **Logical flow** - Sections build on each other coherently
- [ ] **Tables and matrices** - Data presented in scannable formats
- [ ] **Consistent formatting** - Same structure throughout the document
- [ ] **Jargon defined** - Industry terms explained where necessary
- [ ] **Actionable conclusions** - Recommendations tied directly to findings

## Completeness

- [ ] **All sections filled** - No placeholder content remaining
- [ ] **Quantitative data** - Numbers, percentages, and metrics included
- [ ] **Qualitative insights** - Narrative interpretation accompanies data
- [ ] **Visual evidence** - Screenshots, charts, or diagrams where relevant
- [ ] **Recommendations** - Clear next steps provided
- [ ] **Source list** - Complete bibliography with URLs and dates

## Delivery Standards

- [ ] **Naming convention** - File follows squad naming standard
- [ ] **Metadata complete** - Date, author, version, status
- [ ] **Spell-checked** - No spelling errors
- [ ] **Links working** - All referenced URLs accessible
- [ ] **Ready for handoff** - Document standalone (no missing context)

## Scoring

| Score | Rating | Action |
|-------|--------|--------|
| 90-100% | Excellent | Ready for delivery |
| 75-89% | Good | Minor revisions needed |
| 60-74% | Fair | Significant revisions needed |
| Below 60% | Poor | Major rework required |
