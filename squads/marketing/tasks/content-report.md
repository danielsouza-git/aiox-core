# Content Report

```yaml
task:
  id: content-report
  name: "Content Report"
  agent: content-manager
  squad: marketing
  type: content
```

## Proposito

Produzir relatorio mensal de performance organica com metricas de rankings, trafego, comportamento do usuario e analise de ROI do conteudo publicado, fornecendo base para decisoes do proximo ciclo.

## Input

- Dados de analytics (trafego organico, sessoes, bounce rate, tempo na pagina)
- Dados de rankings por keyword (posicoes, variacoes)
- Conteudos publicados no periodo
- Metas de performance organica definidas pelo Head

## Output

- Relatorio mensal de performance organica
- Metricas: rankings, trafego, sessoes, conversoes organicas
- Analise de ROI do conteudo (esforco vs resultado)
- Recomendacoes para o proximo ciclo editorial

## Workflow

### Passo 1: Coletar dados de trafego
Extrair dados de sessoes organicas, pageviews, bounce rate, tempo na pagina e conversoes.

### Passo 2: Acompanhar rankings
Verificar posicoes por keyword, variacoes no periodo e novas keywords rankeando.

### Passo 3: Analisar performance por conteudo
Identificar os conteudos com melhor e pior performance, analisando metricas individuais.

### Passo 4: Calcular ROI
Comparar esforco (horas, custo) com resultado (trafego, leads, conversoes) de cada conteudo.

### Passo 5: Consolidar relatorio
Montar relatorio com visao geral, destaques, problemas e recomendacoes para o proximo ciclo.

## O que faz

- Acompanha rankings e posicoes por keyword
- Analisa trafego organico e comportamento do usuario
- Identifica conteudos com melhor e pior performance
- Calcula ROI do conteudo organico
- Produz relatorio mensal para o Head
- Recomenda ajustes para o proximo ciclo editorial

## O que NAO faz

- NAO reescreve conteudos underperformers (pede a COPY)
- NAO otimiza conteudos publicados (abre nova tarefa de Publish)
- NAO define metas de performance organica sozinho
- NAO faz analise de trafego pago

## Ferramentas

- **Ahrefs** -- Tracking de rankings e backlinks
- **Semrush** -- Auditoria de performance organica
- **Google Sheets** -- Consolidacao de metricas e relatorio
- **YouTube Studio** -- Metricas de videos publicados

## Quality Gate

- [ ] Relatorio mensal entregue ao Head
- [ ] Rankings acompanhados com variacoes documentadas
- [ ] Performance analisada por conteudo individual
- [ ] ROI calculado para conteudos do periodo
- [ ] Recomendacoes para proximo ciclo documentadas
