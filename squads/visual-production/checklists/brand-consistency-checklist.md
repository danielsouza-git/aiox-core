# Brand Consistency Checklist

Ensures visual assets align with brand identity across all touchpoints.

## Color Palette

- [ ] **Primary colors match** - Hex values within tolerance (delta-E < 2)
- [ ] **Color ratios followed** - 60/30/10 (or as specified) distribution
- [ ] **No unauthorized colors** - Every color traceable to brand palette
- [ ] **Tints and shades** - Only approved tint/shade percentages used
- [ ] **Dark mode compliance** - Palette adapts correctly for dark backgrounds

## Typography

- [ ] **Approved fonts only** - No substitutions or similar-looking alternatives
- [ ] **Weight hierarchy correct** - Headlines, body, captions use specified weights
- [ ] **Size scale followed** - Font sizes match the type scale
- [ ] **Line height consistent** - Matches brand typography specifications
- [ ] **Letter spacing correct** - Tracking values match guidelines

## Logo Usage

- [ ] **Correct logo version** - Full, icon, or wordmark as appropriate
- [ ] **Minimum clear space** - Required padding around logo maintained
- [ ] **Minimum size respected** - Logo not smaller than specified minimum
- [ ] **Approved backgrounds** - Logo placed on approved background colors
- [ ] **No modifications** - Logo not rotated, stretched, or recolored

## Photography & Imagery

- [ ] **Style matches direction** - Lighting, mood, subjects align
- [ ] **Color grading consistent** - Same grade applied across set
- [ ] **Subject treatment aligned** - People, products shown per guidelines
- [ ] **No prohibited content** - Avoids imagery flagged in style guide
- [ ] **Quality standards met** - Resolution, sharpness, exposure adequate

## Composition & Layout

- [ ] **Grid alignment** - Elements follow the specified grid system
- [ ] **Whitespace proportions** - Spacing matches brand rhythm
- [ ] **Visual hierarchy clear** - Most important element draws attention first
- [ ] **Consistent margins** - Edge spacing uniform across assets
- [ ] **Balance and alignment** - Elements properly balanced in composition

## Motion & Animation

- [ ] **Timing matches brand** - Duration and easing per brand motion guide
- [ ] **Motion style consistent** - Same animation language across pieces
- [ ] **Reduced-motion provided** - Alternative for prefers-reduced-motion
- [ ] **No excessive motion** - Animation purposeful, not decorative

## Cross-Channel Consistency

- [ ] **Web assets match social** - Same visual language across channels
- [ ] **Print matches digital** - Color reproduction consistent
- [ ] **Campaign cohesion** - All campaign pieces feel unified
- [ ] **Template adherence** - Follows established layout templates

---

## Scoring

| Score | Rating | Action |
|-------|--------|--------|
| 90-100% | Brand-aligned | Approved for use |
| 75-89% | Minor deviations | Fix before external use |
| 60-74% | Significant gaps | Rework required |
| Below 60% | Off-brand | Full redesign needed |
