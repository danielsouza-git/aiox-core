# Social Media Checklist

**Reference:** NFR-7.1
**Items:** 6
**Used by:** qa-reviewer

## Checklist

- [ ] **Platform Dimensions** - Correct dimensions for target platform (see table below)
- [ ] **Text Readability** - Text is legible, proper contrast, not cut off by UI elements
- [ ] **Brand Colors** - Primary/secondary brand colors present and correct
- [ ] **CTA Visibility** - Call-to-action is clear, visible, and compelling
- [ ] **Safe Zone** - Important content within safe zone (not cut off by profile pics, icons)
- [ ] **Hashtags** - Appropriate hashtags (niche/medium/broad mix, no banned tags)

## Platform Dimensions Reference

| Platform | Feed | Stories | Carousel |
|----------|------|---------|----------|
| Instagram | 1080x1080, 1080x1350 | 1080x1920 | 1080x1350 |
| Facebook | 1200x630 | 1080x1920 | 1080x1350 |
| LinkedIn | 1200x627 | 1080x1920 | 1080x1350 |
| X/Twitter | 1600x900 | - | - |
| YouTube | 1280x720 (thumbnail) | - | - |
| TikTok | - | 1080x1920 | - |
| Pinterest | 1000x1500 | 1080x1920 | - |

## Safe Zones

- **Instagram Feed:** 30px margin from edges
- **Instagram Stories:** 250px from top, 150px from bottom
- **YouTube Thumbnails:** Avoid bottom-right (duration badge)

## Verdict Criteria

| Verdict | Criteria |
|---------|----------|
| **PASS** | All 6 items checked |
| **CONCERNS** | 4-5 items checked, issues are cosmetic |
| **FAIL** | <4 items checked or dimensions wrong |
