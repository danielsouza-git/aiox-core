# Brand Identity Checklist

**Reference:** NFR-7.1
**Items:** 7
**Used by:** qa-reviewer

## Checklist

- [ ] **Logo Usage** - Logo used correctly per guidelines (clear space, minimum size, backgrounds)
- [ ] **Color Palette** - All colors match token definitions (no off-brand colors)
- [ ] **Typography** - Correct fonts and hierarchy applied (headings, body, captions)
- [ ] **Spacing/Grid** - Consistent spacing following 8px grid system
- [ ] **Brand Voice** - Copy aligns with voice guide (tone, vocabulary, forbidden words)
- [ ] **Asset Quality** - All assets meet quality requirements (resolution, format)
- [ ] **File Formats** - Correct formats delivered (SVG for logos, PNG/JPG for images)

## Verdict Criteria

| Verdict | Criteria |
|---------|----------|
| **PASS** | All 7 items checked |
| **CONCERNS** | 5-6 items checked, issues are minor |
| **FAIL** | <5 items checked or critical issues |

## Common Issues

1. **Logo clear space violated** - Ensure minimum clear space around logo
2. **Off-brand color used** - Check hex values against tokens
3. **Wrong font weight** - Verify font-weight matches typography system
4. **Inconsistent spacing** - Use 8px multiples consistently
5. **Voice mismatch** - Review copy against brand vocabulary
