## Purpose

Export static site package deployable to any hosting platform without configuration changes.

## Portability Requirements

```yaml
portability:
  paths:
    type: relative_only
    examples:
      - "./assets/css/styles.css"
      - "../images/logo.png"
    prohibited:
      - "/assets/..." (absolute)
      - "https://..." (external CDN)

  dependencies:
    external: none
    fonts: embedded locally
    icons: embedded or sprite
    js_libraries: bundled

  file_protocol:
    works_on: file:// (double-click index.html)
    no_server_required: true
```

## Export Process

```yaml
steps:
  - step: validate_source
    checks:
      - all_files_exist
      - no_broken_links
      - images_optimized

  - step: fix_paths
    action: convert_all_to_relative
    validate: no_absolute_paths

  - step: embed_externals
    fonts:
      - download_google_fonts
      - convert_to_woff2
      - update_css_references
    icons:
      - embed_or_create_sprite

  - step: optimize_assets
    html:
      - minify
      - inline_critical_css
    css:
      - minify
      - remove_unused
    js:
      - minify
      - bundle
    images:
      - compress
      - convert_to_webp (with fallback)

  - step: generate_metadata
    create:
      - sitemap.xml
      - robots.txt
      - README.txt

  - step: validate_package
    tests:
      - open_via_file_protocol
      - all_pages_navigate
      - images_load
      - fonts_render

  - step: create_zip
    compression: deflate
    include_readme: true
```

## Validation Checks

```yaml
validation:
  structural:
    - index.html exists
    - all internal links resolve
    - no 404 references

  portability:
    - no absolute paths
    - no external dependencies
    - works on file:// protocol

  optimization:
    - images under 500KB each
    - total size under 50MB
    - css/js minified

  seo:
    - sitemap.xml valid
    - robots.txt present
    - meta tags complete
```
