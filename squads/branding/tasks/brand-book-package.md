# brand-book-package

```yaml
task: brandBookPackage()
agent: brand-book-builder
squad: branding
prd_refs: [FR-1.7, NFR-9.1, NFR-9.4]

inputs:
  - name: brand_book_dist
    type: directory
    required: true
    source: brand-book-generate output
  - name: brand_book_pdf
    type: pdf
    required: true
    source: brand-book-pdf output
  - name: tokens_package
    type: zip
    required: true
    source: token-export output

  - name: local_package
    type: zip
    destination: .aiox/branding/{client}/exports/brand-book-package.zip

  - archiver
  - cloudflare-r2
```

## Purpose

Create portable local package of Brand Book that works via index.html without server dependencies.

## Packaging Process

```yaml
steps:
  - step: validate_portability
    checks:
      - no_absolute_paths
      - no_external_urls
      - fonts_embedded
      - images_local
    on_fail: fix_paths

  - step: prepare_directory
    structure: package_contents
    copy:
      - brand_book_dist → brand-book/
      - brand_book_pdf → pdf/
      - tokens_package → tokens/ (extracted)
      - logo_assets → assets/logos/
      - icon_assets → assets/icons/
      - font_files → assets/fonts/

  - step: generate_readme
    template: readme_template
    variables: [client_name, version, date, support_email]

  - step: generate_changelog
    source: version_history

  - step: create_zip
    tool: archiver
    format: zip
    compression: deflate
    output: brand-book-package.zip

  - step: validate_package
    actions:
      - extract_to_temp
      - open_index_html
      - verify_navigation
      - verify_search
      - verify_assets

  - step: upload_to_r2
    if: cdn_enabled
    destination: r2://{client}/brand-book/
    create_signed_url: true
    expiry: 1h
```

## Portability Validation

```yaml
portability_checks:
  paths:
    - pattern: "src=\"/"
      issue: "Absolute path"
      fix: "Use relative ./assets/"

    - pattern: "href=\"http"
      issue: "External URL"
      fix: "Embed locally"

    - pattern: "url(http"
      issue: "External CSS resource"
      fix: "Embed in assets/"

  fonts:
    - check: "@font-face uses local files"
    - check: "WOFF2 format present"
    - check: "Fallback fonts specified"

  javascript:
    - check: "No external imports"
    - check: "No CDN dependencies"
    - check: "Works offline"
```

## Signed URL Generation

signed_url:
