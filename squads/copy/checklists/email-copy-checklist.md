# Email Copy Checklist

Quality checklist for email sequences, newsletters, and campaigns.

## Subject Line

- [ ] **Under 50 characters** - Displays fully on mobile
- [ ] **Creates curiosity** - Reader wants to know more
- [ ] **No spam triggers** - Avoids FREE, !!!, ALL CAPS
- [ ] **Personalization** - Uses name or relevant data where appropriate
- [ ] **A/B variants ready** - 2-3 alternatives prepared
- [ ] **Preview text optimized** - First line supports subject

## Opening Hook

- [ ] **First line hooks** - Compelling reason to keep reading
- [ ] **No "Hi [Name]" only** - Greeting isn't the entire first line
- [ ] **Continues subject promise** - Delivers on curiosity gap
- [ ] **Personal tone** - Reads like 1:1 communication
- [ ] **Mobile preview optimized** - First 35-50 chars compelling

## Body Copy

- [ ] **Single focus** - One main idea per email
- [ ] **Scannable format** - Short paragraphs, whitespace
- [ ] **Conversational tone** - Reads like a friend writing
- [ ] **Value delivered** - Reader gains something from reading
- [ ] **Story elements** - Narrative keeps interest
- [ ] **Appropriate length** - Matches email type and audience

## Call to Action

- [ ] **Single primary CTA** - One clear next step
- [ ] **CTA repeated** - Appears 2-3 times in longer emails
- [ ] **Link text descriptive** - "Read the guide" not "Click here"
- [ ] **Button for primary CTA** - If HTML email
- [ ] **PS line with CTA** - Reinforcement at the end

## Sequence Flow (if applicable)

- [ ] **Logical progression** - Each email builds on previous
- [ ] **Timing appropriate** - Spacing between emails makes sense
- [ ] **Story arc** - Sequence tells a cohesive story
- [ ] **Escalating value** - Each email adds something new
- [ ] **Natural close** - Sequence ends with clear resolution

## Personalization

- [ ] **Merge fields tested** - All variables work correctly
- [ ] **Fallback values** - Default text if data missing
- [ ] **Segmentation applied** - Content matches segment
- [ ] **Behavioral triggers** - Dynamic content where appropriate
- [ ] **Tone matches relationship** - New vs. existing subscribers

## Deliverability

- [ ] **Clean HTML** - No excessive code or formatting
- [ ] **Image/text balance** - Not image-heavy
- [ ] **Alt text for images** - Displays if images blocked
- [ ] **Unsubscribe link** - Present and working
- [ ] **From name recognizable** - Recipient knows who it's from
- [ ] **Reply-to monitored** - Real inbox that gets checked

## Mobile Optimization

- [ ] **Readable on mobile** - Single column, adequate font
- [ ] **Links tappable** - Enough spacing between links
- [ ] **Images responsive** - Scale properly on mobile
- [ ] **CTA buttons large** - Easy to tap (44x44 px min)
- [ ] **Preview tested** - Checked on actual mobile device

## Compliance

- [ ] **CAN-SPAM compliant** - Physical address, unsubscribe
- [ ] **GDPR compliant** - If sending to EU
- [ ] **Permission-based** - Recipients opted in
- [ ] **Honest subject line** - Matches email content
- [ ] **Clear sender identity** - Who is sending the email

## Pre-Send Checks

- [ ] **Proof read** - No typos or errors
- [ ] **Links tested** - All links work and track correctly
- [ ] **Personalization tested** - Send test with real data
- [ ] **Segment verified** - Sending to correct list
- [ ] **Schedule confirmed** - Time and date correct
- [ ] **Fallback checked** - What happens if something fails

---

## Email Type Guidelines

| Type | Length | CTAs | Frequency |
|------|--------|------|-----------|
| Welcome | 150-250 words | 1 | Immediate |
| Newsletter | 300-500 words | 2-3 | Weekly/bi-weekly |
| Promotional | 200-400 words | 1-2 | As needed |
| Nurture | 200-350 words | 1 | 2-3 days apart |
| Transactional | 100-200 words | 1 | Triggered |

## Scoring

| Score | Rating | Send Readiness |
|-------|--------|----------------|
| 90-100% | Send ready | Full confidence |
| 80-89% | Almost ready | Minor fixes needed |
| 70-79% | Needs work | Significant issues |
| Below 70% | Not ready | Do not send |
