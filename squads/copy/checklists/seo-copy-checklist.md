# SEO Copy Checklist

Quality checklist for blog posts, pillar content, and SEO-optimized articles.

## Keyword Strategy

- [ ] **Primary keyword identified** - Main term to rank for
- [ ] **Secondary keywords listed** - 3-5 supporting terms
- [ ] **Search intent matched** - Content matches what searchers want
- [ ] **Keyword difficulty assessed** - Realistic ranking potential
- [ ] **Long-tail variations** - Related phrases included naturally

## Title & Meta

- [ ] **Title includes keyword** - Primary keyword in title
- [ ] **Title length 50-60 chars** - Displays fully in SERP
- [ ] **Meta description written** - 150-160 characters
- [ ] **Meta includes keyword** - Natural keyword placement
- [ ] **Meta has CTA** - Encourages click-through
- [ ] **URL slug optimized** - Short, keyword-rich, readable

## Content Structure

- [ ] **H1 tag unique** - One H1 with primary keyword
- [ ] **H2/H3 hierarchy** - Logical heading structure
- [ ] **Keywords in headings** - Natural placement in H2s
- [ ] **Table of contents** - For longer content (2000+ words)
- [ ] **Intro hooks reader** - First 100 words compelling
- [ ] **Conclusion summarizes** - Key points reinforced

## On-Page SEO

- [ ] **Keyword density 1-2%** - Natural, not stuffed
- [ ] **Keywords in first 100 words** - Early placement
- [ ] **LSI keywords included** - Related terms throughout
- [ ] **Image alt text** - Descriptive, includes keywords where relevant
- [ ] **Internal links 3-5** - Links to related content
- [ ] **External links 2-3** - Links to authoritative sources

## Content Quality

- [ ] **Comprehensive coverage** - Topic fully addressed
- [ ] **Unique angle** - Something competitors don't have
- [ ] **Up-to-date information** - Current data and examples
- [ ] **Original research/data** - If applicable
- [ ] **Expert quotes/citations** - Credibility markers
- [ ] **Actionable takeaways** - Reader can apply what they learned

## Readability

- [ ] **Flesch reading ease 60+** - Easy to read
- [ ] **Short sentences** - Average 15-20 words
- [ ] **Short paragraphs** - 2-4 sentences max
- [ ] **Bullet points used** - Break up long sections
- [ ] **Bold key points** - Scannable formatting
- [ ] **Transition words** - Smooth flow between sections

## Media & Visuals

- [ ] **Featured image** - Compelling, relevant hero image
- [ ] **Images throughout** - Break up text every 300-500 words
- [ ] **Images optimized** - Compressed, fast loading
- [ ] **Image file names** - Descriptive, keyword-rich
- [ ] **Infographics/charts** - Where data supports
- [ ] **Video embedded** - If relevant and available

## User Experience

- [ ] **Fast page load** - Under 3 seconds
- [ ] **Mobile responsive** - Reads well on all devices
- [ ] **No intrusive popups** - Minimal interruptions
- [ ] **Easy navigation** - Clear path to related content
- [ ] **Social sharing buttons** - Easy to share
- [ ] **Comments enabled** - If appropriate for site

## Technical SEO

- [ ] **Schema markup** - Article schema implemented
- [ ] **Canonical URL set** - Prevents duplicate content
- [ ] **No broken links** - All links verified
- [ ] **Image dimensions specified** - Prevents layout shift
- [ ] **Lazy loading images** - For performance
- [ ] **Core Web Vitals passed** - LCP, FID, CLS good

## E-E-A-T Signals

- [ ] **Author bio included** - Establishes expertise
- [ ] **Author credentials** - Relevant qualifications shown
- [ ] **Sources cited** - Claims backed by evidence
- [ ] **Updated date shown** - Freshness indicator
- [ ] **Trust signals** - About page, contact info accessible

## Pre-Publish Checks

- [ ] **Spelling/grammar** - Error-free content
- [ ] **Links tested** - All links work
- [ ] **Images display** - All media loads correctly
- [ ] **Mobile preview** - Checked on actual device
- [ ] **SEO tool audit** - Yoast/RankMath/Surfer score
- [ ] **Competitor comparison** - More comprehensive than top results

## Content Length Guidelines

| Type | Word Count | Keywords | Internal Links |
|------|------------|----------|----------------|
| Blog post | 1500-2500 | 3-5 | 3-5 |
| Pillar page | 3000-5000 | 8-12 | 8-15 |
| How-to guide | 2000-3500 | 5-8 | 5-8 |
| List post | 1500-3000 | 4-6 | 4-7 |
| News/update | 500-1000 | 2-3 | 2-3 |

| 80-89% | Good | Minor optimizations needed |
| 70-79% | Fair | Significant SEO gaps |
| Below 70% | Poor | Major revisions required |
