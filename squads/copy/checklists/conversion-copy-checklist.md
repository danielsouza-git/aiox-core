# Conversion Copy Checklist

Quality checklist for landing pages, sales pages, and conversion-focused copy.

## Above the Fold

- [ ] **Headline grabs attention** - Stops the scroll, creates curiosity
- [ ] **Subheadline clarifies** - Expands on headline, adds context
- [ ] **Hero image/video** - Supports the message visually
- [ ] **Primary CTA visible** - Action button above the fold
- [ ] **Value prop clear** - Reader knows the offer in 5 seconds
- [ ] **No navigation distractions** - Minimal links away from page

## Problem/Pain Section

- [ ] **Problem clearly stated** - Reader recognizes their situation
- [ ] **Emotional resonance** - Copy connects with feelings
- [ ] **Agitation appropriate** - Pain amplified without being manipulative
- [ ] **Specificity** - Uses specific scenarios, not vague claims
- [ ] **"You" language** - Speaks directly to the reader

## Solution Section

- [ ] **Solution introduced clearly** - What you're offering
- [ ] **Bridge from problem** - Logical transition from pain to solution
- [ ] **Unique mechanism** - What makes this solution different
- [ ] **Benefits over features** - Focus on outcomes, not specs
- [ ] **Transformation shown** - Before/after clearly illustrated

## Proof Elements

- [ ] **Testimonials included** - Real customer stories
- [ ] **Specificity in proof** - Numbers, names, details
- [ ] **Variety of proof** - Different types (testimonials, data, awards)
- [ ] **Believable claims** - Nothing that triggers skepticism
- [ ] **Visual proof** - Screenshots, photos, videos where applicable

## Offer Section

- [ ] **Offer clearly stated** - What exactly they're getting
- [ ] **Value stack** - All components listed
- [ ] **Price justified** - Value demonstrated before price
- [ ] **Bonuses included** - Additional value items
- [ ] **Guarantee present** - Risk reversal element

## Urgency & Scarcity

- [ ] **Real urgency** - Not manufactured or fake
- [ ] **Clear deadline** - Specific date/time if applicable
- [ ] **Consequence of delay** - What happens if they wait
- [ ] **Scarcity justified** - Reason for limited availability
- [ ] **Ethical implementation** - No dark patterns

## CTA Optimization

- [ ] **Button text action-oriented** - "Get Access Now" not "Submit"
- [ ] **Multiple CTAs** - Repeated throughout long pages
- [ ] **Button stands out** - Contrasting color, visible size
- [ ] **Friction reduced** - Minimal form fields
- [ ] **Microcopy supports** - Text around button reduces anxiety

## Objection Handling

- [ ] **FAQ section** - Common questions answered
- [ ] **Money objection** - Price justified or payment options
- [ ] **Time objection** - Easy/quick to implement
- [ ] **Trust objection** - Credibility established
- [ ] **"Not for me" objection** - Clear who it's for

## Mobile Optimization

- [ ] **Readable on mobile** - Font size, spacing appropriate
- [ ] **CTAs thumb-friendly** - Easy to tap
- [ ] **Images optimized** - Load quickly, display properly
- [ ] **Forms mobile-friendly** - Easy to complete on phone
- [ ] **No horizontal scroll** - Everything fits screen width

## Technical

- [ ] **Page loads fast** - Under 3 seconds
- [ ] **Links work** - All CTAs tested
- [ ] **Form submits** - Tested end to end
- [ ] **Tracking in place** - Analytics, pixels ready
- [ ] **A/B test ready** - Variants prepared if needed

## Scoring

| Score | Rating | Conversion Readiness |
|-------|--------|---------------------|
| 90-100% | Launch ready | Full confidence |
| 80-89% | Almost ready | Minor tweaks needed |
| 70-79% | Needs work | Significant gaps |
| Below 70% | Not ready | Major revisions required |
