# First Value

```yaml
task:
  id: first-value
  name: "First Value"
  agent: onboarding-specialist
  squad: customer-success
  type: onboarding
```

## Proposito

Garantir que o cliente tenha sua primeira vitoria concreta com o produto -- o momento em que percebe valor real na compra. Esse momento e critico para a retencao de longo prazo e deve acontecer o mais rapido possivel apos o setup.

## Input

- Cliente com conta configurada (Setup Account completo)
- Necessidades mapeadas pelo Closer durante a venda
- Metricas de uso inicial do produto

## Output

- Primeira vitoria do cliente confirmada e documentada
- Validacao de que o cliente esta usando o produto corretamente
- Feedback inicial coletado sobre a experiencia
- Registro no CRM com status "Onboarding - First Value Achieved"

## Workflow

### Passo 1: Identificar o quick win
Com base nas necessidades mapeadas na venda, definir qual e a primeira vitoria mais rapida e impactante.

### Passo 2: Guiar o cliente ate a vitoria
Acompanhar o cliente no uso do produto ate que ele atinja o resultado esperado.

### Passo 3: Validar uso correto
Verificar que o cliente esta usando o produto da forma correta e nao apenas "deu sorte".

### Passo 4: Coletar feedback
Perguntar sobre a experiencia, o que foi facil, o que foi dificil e o que pode melhorar.

### Passo 5: Documentar e celebrar
Registrar a primeira vitoria e comunicar ao cliente que ele esta no caminho certo.

## O que faz

- Garante a primeira vitoria concreta do cliente
- Valida que o uso esta correto e sustentavel
- Coleta feedback inicial sobre a experiencia
- Documenta o resultado alcancado

## O que NAO faz

- NAO resolve problemas tecnicos (escala para Suporte)
- NAO configura funcionalidades avancadas
- NAO faz o handoff para CS (isso e do Handoff to CS)

## Ferramentas

- **Zoom** -- Call de acompanhamento
- **WhatsApp** -- Follow-up rapido
- **CRM** -- Registro de atividades e marco de first value
- **Notion** -- Documentacao de feedback

## Quality Gate

- Threshold: >70%
- Cliente alcancou resultado concreto com o produto
- Uso validado como correto e sustentavel
- Feedback inicial coletado e documentado
- Status "First Value Achieved" registrado no CRM
