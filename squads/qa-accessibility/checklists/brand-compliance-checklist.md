# Brand Compliance Checklist

Comprehensive brand guideline compliance checklist for all deliverables.

## Logo Usage

- [ ] **Correct variant** - Appropriate logo variant for context (primary, secondary, icon)
- [ ] **Clear space** - Minimum exclusion zone around logo respected
- [ ] **Minimum size** - Logo meets minimum dimension requirement
- [ ] **Color variant** - Correct color version (full color, mono, reversed)
- [ ] **Background** - Logo visible and appropriate on its background
- [ ] **No modifications** - Logo not stretched, rotated, or altered
- [ ] **No effects** - No shadows, gradients, or filters applied
- [ ] **Aspect ratio** - Logo proportions maintained

## Color Palette

- [ ] **Primary colors** - Used correctly for primary elements (CTAs, headings)
- [ ] **Secondary colors** - Applied to appropriate secondary elements
- [ ] **Accent colors** - Used sparingly for highlights and emphasis
- [ ] **Neutral colors** - Body text, backgrounds, borders use neutral scale
- [ ] **Semantic colors** - Success/warning/error use correct approved colors
- [ ] **Exact hex match** - All colors match approved hex values exactly
- [ ] **No off-palette colors** - No unauthorized colors used anywhere
- [ ] **Gradient compliance** - Gradients follow approved specifications

## Typography

- [ ] **Approved font families** - Only sanctioned typefaces used
- [ ] **Font weights** - Only approved weights loaded and used
- [ ] **Type scale** - Font sizes follow the approved scale
- [ ] **Line heights** - Leading matches specifications per element
- [ ] **Letter spacing** - Tracking values match where specified
- [ ] **Text transforms** - Uppercase, lowercase applied per guidelines
- [ ] **Font loading** - WOFF2 format, preloaded, swap fallback
- [ ] **Hierarchy** - Visual hierarchy clear and consistent

## Spacing & Layout

- [ ] **Grid system** - Layout follows the defined grid system
- [ ] **Spacing scale** - Margins/padding use the spacing scale values
- [ ] **Component gaps** - Inter-component spacing consistent
- [ ] **Section spacing** - Section separations match guidelines
- [ ] **Whitespace** - Appropriate breathing room per brand style
- [ ] **Alignment** - Elements align to grid and each other

## Imagery & Iconography

- [ ] **Image style** - Photography/illustration style matches brand
- [ ] **Image treatment** - Filters, overlays, crops per guidelines
- [ ] **Icon set** - Icons from approved icon set
- [ ] **Icon sizing** - Icons at approved sizes
- [ ] **Icon color** - Icons use approved palette colors
- [ ] **Illustration style** - Consistent with brand illustration guidelines

## Tone & Messaging

- [ ] **Voice consistent** - Copy matches brand voice characteristics
- [ ] **Approved vocabulary** - Uses approved terms, avoids forbidden words
- [ ] **Messaging hierarchy** - Primary/secondary messages prioritized correctly
- [ ] **Tagline usage** - Tagline used correctly (if applicable)
- [ ] **Legal compliance** - Required disclaimers and notices present

## Scoring

| Score | Rating | Action |
|-------|--------|--------|
| 95-100% | Fully Compliant | Approved for delivery |
| 85-94% | Minor Violations | Fix before delivery |
| 70-84% | Significant Violations | Revisions required |
| Below 70% | Non-Compliant | Major rework needed |
